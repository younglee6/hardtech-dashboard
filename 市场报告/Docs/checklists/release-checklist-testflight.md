# TestFlight Release Checklist

## Build & Version
- [ ] Increment build number and version.
- [ ] Archive build succeeds without warnings that block release.

## App Metadata
- [ ] App name, subtitle, and description updated.
- [ ] Screenshot set prepared.
- [ ] App icon and launch assets verified.

## Compliance
- [ ] Privacy usage descriptions match actual permissions.
- [ ] Export compliance answered.

## Quality Gate
- [ ] Core user journey tested end-to-end.
- [ ] Crash-free smoke test on physical device.
- [ ] Known issues documented.

## Submission
- [ ] Internal tester group selected.
- [ ] Build notes include test focus and known limitations.
