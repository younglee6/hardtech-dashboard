# Week 01 Setup Checklist (iOS Native)

## Tooling
- [ ] Install latest stable Xcode.
- [ ] Confirm Apple Developer account access.
- [ ] Configure Git identity (`git config user.name`, `git config user.email`).

## Device & Signing
- [ ] Connect physical iPhone and trust this Mac.
- [ ] Enable automatic signing in Xcode.
- [ ] Verify Bundle Identifier uniqueness.

## Build Validation
- [ ] Run app on simulator.
- [ ] Run app on real device.
- [ ] Archive succeeds once.

## TestFlight Dry Run
- [ ] Create app record in App Store Connect.
- [ ] Upload one test build.
- [ ] Confirm internal tester can see build.

## Learning Artifacts
- [ ] Save setup notes in `Docs/weekly/week-01-review.md`.
- [ ] Record setup blockers and fixes.
