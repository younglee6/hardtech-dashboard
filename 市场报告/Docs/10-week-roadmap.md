# 10-Week Execution Roadmap (3-4h/week)

## Week 1
- Goal: Environment setup + run first SwiftUI screen.
- Deliverable: App launches and shows Habit list shell.

## Week 2
- Goal: List -> Detail -> Add flow complete.
- Deliverable: Demo video (30-60s) for full navigation loop.

## Week 3
- Goal: Add local persistence with UserDefaults store.
- Deliverable: Data remains after restart.

## Week 4
- Goal: Unified error/loading state handling.
- Deliverable: Empty/loading/error/success states all visible and testable.

## Week 5
- Goal: Lock PRD and finish 60% MVP.
- Deliverable: Core check-in flow works.

## Week 6
- Goal: Reach 70%+ MVP completeness.
- Deliverable: Main user path stable with basic validation.

## Week 7
- Goal: Add unit tests for use cases and view model.
- Deliverable: Core tests pass and known issues documented.

## Week 8
- Goal: Release readiness polish.
- Deliverable: Icon/metadata/privacy notes and bug fixes complete.

## Week 9
- Goal: TestFlight submission rehearsal.
- Deliverable: Archive steps documented and dry-run complete.

## Week 10
- Goal: Internal testing + final retrospective.
- Deliverable: TestFlight build shared to testers and retrospective finalized.
