# Metrics Framework

## Core Principle

先看北极星指标，再拆一级杠杆，最后下钻到行为事件与人群分层。

## AARRR Metric Tree

1. Acquisition（拉新）
- 新增用户数
- 新增成本（CAC）
- 渠道转化率（点击到注册）
- 有效新客占比（满足激活条件）

2. Activation（激活）
- 首日关键行为完成率（如完成首次核心动作）
- 首日任务完成率
- 首次价值感知时长（TTV）
- 激活漏斗各步骤转化率

3. Retention（留存）
- D1/D7/D30 留存
- 周活/月活（WAU/MAU）
- 回访频次
- 沉默用户占比

4. Revenue（变现）
- 付费转化率
- ARPU/ARPPU
- 续费率
- 退款率

5. Referral（传播）
- 分享率
- 裂变邀请转化率
- 每用户带来新增用户数（K-factor）

## Common Segmentation

- 用户生命周期：新手期、成长期、成熟期、流失风险期
- 渠道：自然、买量、活动、社交传播
- 人群属性：设备、地区、系统版本
- 行为分层：高频/低频、高价值/低价值

## Quick Diagnostic Sequence

1. 比较环比/同比确认异常是否持续
2. 用分群判断是否局部问题
3. 用漏斗确认是入口问题还是流程问题
4. 回到事件层确认具体中断点
